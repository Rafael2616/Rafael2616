## 🚀 Sobre mim

Sou desenvolvedor focado em **Android com Kotlin** e **Jetpack Compose**, sempre buscando criar interfaces fluidas, acessíveis e escaláveis.  
Atualmente também trabalho com **Kotlin Multiplatform (KMP)** para entregar soluções multiplataforma com performance nativa.

Tenho experiência com:
- Google Maps (Android & iOS)
- Processamento e otimização de imagens
- Firebase, autenticação e notificações
- Acesso a arquivos e permissões (Android/iOS)
- Compose Desktop e WASM/JS
- Arquiteturas modernas (MVVM, MVI, Clean Architecture)

Adoro resolver problemas complexos, refinar desempenho e transformar boas ideias em produtos reais.
